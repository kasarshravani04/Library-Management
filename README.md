# 📚 Library Management System

A modern, lightweight web application built to manage book collections efficiently. This project demonstrates frontend CRUD operations, multi-page navigation, and persistent data handling without a backend.

- **Storage:** LocalStorage API (No database/backend required)

---

### ✨ Key Features

- **Multi-Page Interface:** - `Home`: View the full book collection with search capabilities.
  - `Add Book`: A dedicated form to register new titles into the system.
- **Dynamic Search & Filter:** Real-time filtering system that allows users to find books by title or author instantly as they type.
- **Persistent Storage:** Uses `localStorage` to ensure your library data remains saved in the browser even after a page refresh or browser restart.
- **Responsive Design:** A clean, nature-inspired blue aesthetic with a focus on readability and ease of use.

---

### 💾 Technical Highlight: LocalStorage
This project showcases my ability to handle data persistence on the client side. 
- **JSON Handling:** Implemented `JSON.stringify()` for storage and `JSON.parse()` for data retrieval.
- **State Management:** The UI stays in sync with the local data array, ensuring that any "Add" or "Delete" action is immediately reflected on the screen.

---

### 📂 File Structure
- `index.html` - The main dashboard for the book collection.
- `add-book.html` - The interface for adding new books.
- `bookList.js` - Logic for rendering the list and filtering data.
- `bookList.css` - Custom styling for a professional user experience.
